# steps to get started 

## start the backend so we can fetch the data later
- cd backend
- npm i express nodemon mysql2 cors express-session body-parser cookie-parser
- npm start 

## setup the react app
first open a new terminal then : 
- cd client
- npm install
- npm i react-router-dom

- npm start
